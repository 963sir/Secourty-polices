SIR | Security is committed to protecting your privacy while providing advanced moderation and protection features on Discord.

Data Collected: Only essential Discord data is collected for moderation purposes (user IDs, roles, messages for rule enforcement, and server audit logs if necessary).

Data Usage: Collected data is used solely to prevent spam, enforce server rules, monitor links, and generate moderation reports.

Data Storage & Retention: Temporary data like recent messages or logs is stored securely in memory or short-term storage and cleared automatically. Server configuration (roles, filter settings) may be saved locally for functionality.

No Third-Party Sharing: We do not sell or share your personal data with external parties, except for legal compliance if required.

User Rights: You can request information about data the bot has processed or request deletion by contacting us.
